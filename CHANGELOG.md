# Change Log

## 1.0.0 - Initial Release

- Interactive VCD waveform viewer for FPGA/RTL developers
- All features free and unlimited — no license required
- Full signal search with hierarchical module browsing
- Drag-and-drop signal reordering
- Multiple radix formats (Binary, Octal, Hex, Signed, Unsigned, ASCII, UTF-8, Float)
- Configurable keyboard shortcuts
- Theme-aware rendering with PIXI.js WebGL canvas
- Custom editor provider for `.vcd` files
